## 카테고리

- **Python** - 파이썬 문법, 함수, 클래스
- **Git** - Git 사용법, 워크플로우
- **Algorithm** - 알고리즘 문제 풀이
- **Error Log** - 에러 해결 기록

## 로컬에서 실행

```bash
npm install
npm start
```

http://localhost:3000/TWL/jsc/ 에서 확인

## 빌드 및 배포

### GitHub Actions (자동)

`main` 브랜치에 push하면 자동으로 GitHub Pages에 배포됩니다.

### 직접 배포

```bash
npm run build
npm run deploy
```
