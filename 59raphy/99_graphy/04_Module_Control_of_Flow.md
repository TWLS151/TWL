# Modules


## 모듈
- 한 파일로 묶인 변수와 함수의 모음
- 파이썬 파일(.py)
1. > import 모듈명
    
        print(모듈명.변수명)
        print(모듈명.함수명)
    - 단일 모듈에서 권장
2. > from 모듈명 import 변수명, 함수명
    
        print(변수명)
        print(함수명)
    - 서로 다른 모듈에서 import된 변수나 함수의 이름이 같은 경우, 마지막에 import된 것만 유효함
        - 'as 키워드'로 별칭(alias) 부여
        > from 모듈명 import 변수/함수명 as 새로운이름

-  사용자 정의 모듈
    1. 모듈명.py 생성하여 함수 작성
    2. *같은 위치*에 파이썬 파일 생성하고 모듈 import 및 함수 호출

- 파이썬 표준 라이브러리(PSL)
    - 라이브러리 : 다양한 패키지들을 포함하는 상위 개념
    - 라이브러리 > 패키지 > 모듈

### 패키지
- 연관된 모듈(.py)들을 하나의 디렉토리에 모아 놓은 것. 코드 꾸러미
    > from 패키지명.서브패키지명 import 모듈명

        print(모듈명.변수명)
        print(모듈명.함수명)
    - 같은 경로에 있는 패키지부터 적용
- PSL 내부 패키지
    - 설치 없이 바로 impport해서 사용 가능
- 파이썬 외부 패키지
    - pip 사용하여 설치
    - requests 패키지 : 파이썬에서 웹에 요청을 보내고 응답을 받기 위한 패키지
        > $ pip install requests

        > import requests



# Control of Flow


## 제어문
- 코드의 실행 흐름을 제어하는 데 사용되는 구문
- 조건에 따라 코드 블록을 실행하거나 반복적으로 코드를 실행


## 조건문
- 주어진 조건식을 평가하여 해당 조건이 참(True)인 경우에만 코드 블록을 실행하거나 건너뜀

### 'if' statement
- if문
    - 조건은 *표현식*으로 작성
- elif문
    - 여러 개 사용 가능     
- else문
    - 모든 조건들을 만족하지 않으면 실행
    - 없어도 됨
- 복수 조건문
    - 조건식을 위에서부터 *순차적*으로 비교
- 중첩 조건문


## 반복문
- 주어진 코드 블록을 여러 번 반복해서 실행

### 'for' statement
- *반복 가능한(iterable)* 객체의 요소들을 반복
- 반복 횟수가 정해져 있음(주로 반복 가능한 객체 요소의 개수만큼)
    > for 임시변수 in 반복가능객체:

**반복 가능한 객체(iterable)**
- 반복문에서 순회할 수 있는 객체
    - 시퀀스 자료형 : list, tuple, str
    - 비시퀀스 자료형 : dict, set
    - 숫자형은 반복 불가
        - range()로 정수 시퀀스 생성하여 반복
    
- 문자열 순회 : 문자가 반복 변수에 할당되어 반복 수행
- range 순회 : 특정 숫자 범위만큼 반복
- 딕셔너리 순회 : *key를 추출*
- 리스트 순회 : 리스트의 요소가 아닌 *인덱스*로 접근하여 해당 요소를 변경
    > for i in range(len(리스트명)):

        리스트명[i] = 리스트명[i] * 2
- 중첩 리스트(2차원 리스트) 순회
    > for 바깥임시변수 in 리스트명:

        for 안쪽임시변수 in 바깥임시변수:
            print(안쪽임시변수)

### 'while' statement
- 조건이 참(True)인 동안 반복(False가 되면 종료)
- 주로 반복 횟수가 정해지지 않은 경우 사용
- 특정 조건이 만족될 때까지 반복해야 하는 경우 사용
- *반드시 종료 조건이 필요*

### 반복 제어
- 반복 제어 키워드
    - break : 반복 즉시 종료
    - continue : 남은 코드 무시하고 다음 반복 수행
    - 반복 제어문은 반드시 반복문 내에서 사용
    - 중첩 반복문의 경우 키워드가 작성된 코드블록에만 적용
- 빈 코드 블록 키워드
    - pass : 아무 동작도 하지 않음


cf) flag 변수


### map 함수
> map(function, iterable)
- 반복 가능한 데이터구조의 모든 요소에 함수를 적용하고, 그 결과 값들을 map object로 묶어서 반환
- map object : 결과를 하나씩 꺼내 쓸 수 있는 반복 가능한 객체 자료형
    - 리스트나 튜플로 형변환하여 전체 값 확인
- 함수 자리에 lambda 표현식을 넣어서 활용

### zip 함수
> zip[(*iterables)]
- 여러 개의 반복 가능한 데이터 구조를 묶어서 같은 인덱스의 값들을 하나의 *tuple*로 만든 뒤 zip object로 반환
- zip object : 짝지어진 결과(tuple)를 하나씩 꺼내 쓸 수 있는 반복 가능한 객체 자료형
    - 리스트나 튜플로 형변환하여 전체 값 확인
- 반복 가능한 자료형의 길이가 다른 경우 가장 짧은 길이를 기준으로 묶어서 반환
- 언패킹하여 변수에 바로 tuple 요소 할당 가능
- 2차원 리스트의 같은 열 요소를 동시에 조회할 때
    - *전치 행렬*과 실행 결과가 동일

### enumerate 함수
> enumerate(iterable, start=0)
- iterable 객체의 각 요소에 대해 *인덱스와 값을 함께 반환*하는 내장함수
- start에 시작 값 설정(기본 값 0)
