# 1단계: Git Pull - 최신 코드 동기화

작업을 시작하기 전에 항상 최신 코드를 로컬에 동기화해야 합니다.

## 📌 Git Pull이란?

리모트 저장소의 최신 코드를 로컬에 다운로드하고 병합하는 명령어입니다.

## 🔧 실행 방법

### 1. `main` 브랜치로 이동
```bash
git checkout main
```

### 2. 최신 코드 동기화
```bash
git pull origin main
```

## 📝 상세 설명

- `git pull` = `git fetch` + `git merge`
- 리모트(깃허브에 올라와있는)의 최신 변경사항을 로컬(내 컴퓨터)에 반영합니다
- 커밋하지 않은 코드가 있을 경우 충돌이 발생할 수 있으니 주의하세요

## ✅ 완료 확인

```bash
git log -1
```
```bash
git status
```
최신 커밋이 표시되면 성공입니다!

## 💡 팁

- 새 브랜치를 만들기 전에 항상 Pull을 먼저 실행하세요
- 충돌 발생 시 팀원과 상의하세요
