---
# 1. LOCAL
+ 의미 : 함수 안에서 정의 한 변수

# 2. Enclosed
+ Enclosed function locals의 약자로 내부함수에서 자신의 외부 함수의 범위
+ 함수 안에서 함수를 새로 정의할 경우 외부 함수의 변수와 내부 함수의 변수가 다른 스코프를 지니게 되고, 이를 구별하여 인지할 수 있어야 한다.
# 3. Global
+ 의미 : 전역 변수
+ 함수 밖에서 선언된 변수.
함수 밖에서 정의되었더라도. 
global 변수명 
같은 방식으로 바깥의 변수에 접근이 가능합니다.

# 4. Built-in
의미 : Built-in의 약자로 open, range와 같은 파이썬 내장 함수
