## 참고

### 딕셔너리의 확장: defaultdict
- 내장 모듈 collections에서 제공하는 딕셔너리의 확장판
- 딕셔너리에 존재하지 않는 키를 조회할 때, *자동으로 '기본값'을 생성*해주는 자료구조
- KeyError 걱정 없이 값을 수정하거나 추가
1. 숫자 세기 : defaultdict(int)
    - 키가 없으면 기본값을 0으로 설정
2. 그룹핑/리스트 모으기 : defaultdict(list)
    - 키가 없으면 기본값을 빈 리스트 []로 설정
- 조회만 해도 키가 생성되므로, 단순히 키가 있는지만 확인할 때는 get 메서드 사용
- 호출할 때마다 값을 넣어줘야 하는 .setdefault()와 달리, 객체를 생성할 때 기본값을 한 번만 설정하면 됨

### 파이썬 문법 규격
- BNF : 여러 프로그래밍 언어의 문법을 공통적으로 표현하기 위한 문서 표기법
- EBNF : 확장 BNF 표기법
    - 메타 기호
        - [] : 선택적 요소
        - {} : 0번 이상 반복
        - () : 그룹화
