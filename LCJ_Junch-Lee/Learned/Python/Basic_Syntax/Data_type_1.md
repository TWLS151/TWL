## Sequence Type
### str, list, tuple, range, ...

TIP
- 각 데이터는 기차 칸 처럼 서로 연결됨
- 모든 칸에는 0번부터 시작하는 고유한 번호(인덱스)가 존재

&nbsp;

### 시퀀스 타입의 5가지 공통 특징

- (1) 순서 : 값들이 순서대로 저장. (정렬 X)

- (2) 인덱싱 : 고유 번호 존재, 인덱스를 통한 참조 및 수정 가능
&nbsp;

- (3) 슬라이싱 : 원하는 부분만을 잘라내 사용 가능
&nbsp;

- (4) 길이 : `len()` 함수를 사용하여 저장된 값의 개수(길이)를 구할 수 있음 

- (5) 반복 : 반복문을 사용하여 각 값을 하나씩 순서대로 꺼내 사용할 수 있음 

&nbsp;

## 1. str (문자열)

: 문자들의 순서가 있는 **변경 불가능**한 시퀀스 자료형

&nbsp;

### 1. 문자열의 활용
   
- #### 이스케이프 문자열 : `\`를 사용

    - `\n`: 개행, `\f` : 탭(Tab) 등 다양한 기능 제공

- #### f - string (파싱)

- 문자열 시작 전 'f'접두어를 붙이고 삽입할 부분(표현식)을 중괄호 {}로 감싸줌 

- 문자열 내에서 변수, 함수의 반환값을 바로 활용할 수 있어 유용

```python
print("문자열에서 \\사용하기")

# 문자열에서 \ 사용하기

T = 1
result = 100

print(f"정답 출력 : # {T} : {result}")

# 정답 출력 : # 1 : 100
```

&nbsp;

### 2. 시퀀스로서의 문자열 활용 

**인덱스(Index) : 시퀀스 자료형에서 각 값의 위치를 식별하기 위해 부여된 고유한 번호**


- 파이썬은 **음수 인덱스**를 지원 

- 슬라이싱 방법

```python
# 1. start, end를 지정 

my_str = hello

my_str[2:4] 

print(my_str[2:4]) # ll 

# 2. start를 생략

my_str[:3]

print(my_str[:3]) #hel 

# 3. end를 생략

print(my_str[3:]) # lo

# 4. step * 음수 인덱스

print(my_str[::-1]) # olleh
```

## 추가 : 재할당과 가변

- ### 재할당(Reassignment)

: 변수가 가리키는 객체(메모리 주소) 자체를 바꾸는 것

```python

x = 10

x = 20 # x 라는 이름표를 떼어서 숫자 20에 다시 붙임 (재할당)
```

- ### 가변성(Mutability)

: 생성된 객체 내부의 요소를 직접 수정할 수 있는지를 나타내는 속성

- `list`, `dict`, `set` 등이 포함됨

```python

my_list = [1, 2, 3]

my_list[0] = 99

print(my_list)

# [99, 2, 3]

# 재할당이 아니라 '내부 값' 자체를 수정(가변성)
```
