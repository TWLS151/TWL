# 부녀회장이 될 테야!!

자존심 회복 프로젝트(알고리즘 실력 다시 끌어올리기)를 위해서 오늘은 DP문제를 풀었습니다.


걍 있는 값에 대해서 모두 더하면 끝! 뭐 그런 느낌입니다. 


``` python
# 2775 -  정리해서 올려드립니다.
A = int(input())
#인풋 받기
for i in range(A):
    K = int(input()) # 1 층 수 받기
    N = int(input()) # 3 호수 받기
    L = []
    L.append(list(range(1,N+1))) #0층 사람 수 삽입
    # print(L)
    for i in range(K): #==>층수 # N개 층까지 돌면서 
        L.append([]) # 층 만들고
        for j in range(N): #==> 그 층까지 숫자를 다 더하세요!
            L[-1].append(sum(L[-2][:j+1])) #전전 층 호 까지 다 조회해서 더하기!

    print(L[-1][-1])

```


그래도 이번엔 이렇게 풀어야지! 가 들어맞아서 좋았다.
