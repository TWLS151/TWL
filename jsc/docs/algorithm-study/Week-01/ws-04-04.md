## Issue-1
- S
  - WS-04
  - 26.01.25
- T
  - API_URL을 for 문제에 `{i}`를 사용해서 자동 반복 입력
- A
  - 주소 앞에 'f'를 붙여서 f-string으로 변환
- R
  - '  ' 안에 `{i}`로 변수를 처리하더라도 문자열로 인식을 해서 의미가 없다.
  앞으로 이런상황에서 앞에 'f'를 붙여보자.

- 에러 났던 코드
```py
import requests
from pprint import pprint as print

dummy_data = []
for i in range(1,11):
# 무작위 유저 정보 요청 경로
    API_URL = f'https://jsonplaceholder.typicode.com/users/{i}'
    # API 요청
    response = requests.get(API_URL)
    # JSON -> dict 데이터 변환
    parsed_data = response.json()
    dummy_data.append(parsed_data['username'])

# # 응답 데이터 출력
# print(response)

# # 변환 데이터 출력
print(dummy_data)
# # # 변환 데이터의 타입
# print(type(parsed_data))

# 특정 데이터 출력
# print(parsed_data['username'])
```
