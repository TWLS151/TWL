### **자료구조1**
- Data Structure
  - **메서드**
    - 데이터.메서드()의 형태
    - 클래스 내부에 정의되는 함수
    - 모듈.함수()와 구분 (내장함수도 구분)
    - 학습 위해서 기능을 직접 사용
- 공통 시퀀스 메서드
  - 자주 나오는 메서드 정리
- 불변 시퀀스 메서드
  - [,] -> 선택 인자(문서적 표현, 코드 내에 작성하면 에러 발생, 설명 용도)
  - .split -> 단어들의 리스트를 반환
    - SWEA 에서 자주 등장 
- 가변 시퀀스 메서드 
   - .append()
   - .pop()
   - (리스트 메서드 정리)    
     - list()와 []의 차이점에 대해서 알게 되었다.
- 복사
  - 얕은 복사
    - [:], copy(), list()
    - 다차원으로 가는 순간 원본 객체에 영향
  - 깊은 복사
    - `import copy`, `.deepcopy()`
    - 얕은 복사의 문제를 모두 해결
    - 원본을 유지한 채로 문제를 해결해야 하는 경우 활용
- List Comprehension
  - 안에 들어가는 표현식이 여러줄이면 못쓴는 상황이 올 수 있음
  - 해석과 활용을 위한 이해는 해두자. 남용 금지
- 메서드 체이닝
  - 여러 메서드를 동시에 실행할 수 있음
  - 모든 메서드가 체이닝을 지원하지 않음
