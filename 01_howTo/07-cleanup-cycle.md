# 7단계: 로컬 정리 및 사이클 반복

병합 후 로컬 저장소를 정리하고 다음 작업을 준비합니다.

## 🧹 로컬 정리

### 1. 현재 상태 확인
```bash
git status
```

### 2. main 브랜치로 이동
```bash
git checkout main
```

### 3. 최신 코드 동기화
```bash
git pull origin main
```

### 4. 로컬(내 컴퓨터에 있는) 브랜치 삭제
```bash
# 단일 브랜치 삭제
git branch -d learned/your-feature-name

# 여러 브랜치 삭제
git branch -D feature/old-branch
```

## 📋 사이클 반복 준비

모든 정리가 끝났다면 다음 작업을 시작할 준비가 됐습니다!

## 💡 팁과 트러블슈팅

### Q: 커밋을 실수로 잘못 했어요
```bash
git reset HEAD~1
```

### Q: 브랜치를 삭제했는데 복구하고 싶어요
```bash
git reflog
git checkout 커밋해시
```

### Q: 병합 중 충돌이 났어요
```bash
# 충돌 파일 수정 후
git add .
git commit -m "충돌 해결"
git push
```

### Q: 푸시 전에 커밋을 수정하고 싶어요
```bash
git commit --amend -m "새로운 메시지"
git push -f
```
