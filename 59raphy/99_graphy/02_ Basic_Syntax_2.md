# Basic Syntax 2


## 형변환

### 암시적 형변환
- 정수 + 실수  # 정수 -> 실수
- 불리언 + 정수  # 불리언 -> 정수
    - True == 1, False == 0
- 불리언 + 불리언  # 불리언 -> 정수

### 명시적 형변환
- str -> int
    - 형식에 맞는 숫자만 가능
        > print(int(3.14))  # ValueError
- int -> str
    - 모두 가능
- 컬렉션 간 형변환
    - str, list, tuple, range, set, dict
    - range로 변환 : 불가
    - dict로 변환 : 불가
    - dict -> list/tuple/set : key만 변환

### 비교 연산자
- '<=' : 이하
- '>=' : 이상
- == 연산자 : 값이 같은가
- is 연산자 : 객체(메모리 주소)가 같은가
    - 싱글턴 객체를 비교할 때 사용
        > 특정 값에 대해 파이썬 전체에서 단 하나의 객체만 생성되어 재사용되는 특별한 객체

        > None, True, False

### 논리 연산자
- and(논리곱) : 모두 T면 T로 평가
    - 하나라도 F면 F
- or(논리합) : 하나라도 T면 T로 평가
    - 모두 F면 F
- not(논리부정) : 단일 피연산자를 부정
    - print(not 0)  # True

**단축 평가**
- 거짓으로 취급되는 값들
    > False, 0, 빈 문자열 "", 빈 리스트 [], None 등
- and 단축 평가 : 하나라도 F면 F
    - 처음 만나는 *거짓 값* 반환
    - 모두 참이면 맨 마지막 참 값 반환
- or 단축 평가 : 하나라도 T면 T
    - 처음 만나는 *참 값* 반환
    - 모두 거짓이면 맨 마지막 거짓 값 반환
