# TWL (Today We Learned)

15기 서울 1반의 모두가 TIL을 공유해서 올리는 Today We Learned!!

## 📋 프로젝트 소개

15기 서울 1반 팀원들이 매일 배운 내용을 공유하고 기록하는 협업 저장소입니다.

## 🎯 프로젝트 목표

- 팀원 간 학습 내용 공유
- 일일 학습 기록 관리
- 상호 피드백을 통한 학습 품질 향상

## 📁 디렉토리 구조

```
TWL/
├── README.md
├── howTo/
├── templates/
├── [팀원이름]/
│   └── 날짜/
│       └── 주제.md
└── ...
```
위의 형식을 차용하여 개인 이름(개인 깃허브 아이디 명)으로 폴더를 제작하고 그 안에서 TIL들을 작성해주시면 됩니다!
