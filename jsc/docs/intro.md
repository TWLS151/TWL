# Welcome to My TIL

**TIL (Today I Learned)** 은 매일 배운 것을 기록하는 공간입니다.

## 왜 TIL을 쓰나요?

- 📝 **기록의 힘**: 배운 것을 글로 정리하면 더 오래 기억됩니다
- 🔍 **나만의 위키**: 필요할 때 빠르게 찾아볼 수 있습니다
- 📈 **성장의 증거**: 꾸준한 기록이 모여 실력이 됩니다

## 작성 원칙

1. **간결하게** - 60줄 이하로 핵심만
2. **실용적으로** - 실제 사용할 수 있는 내용 위주
3. **꾸준하게** - 완벽하지 않아도 매일 조금씩

## 시작하기

사이드바에서 카테고리를 선택하여 문서를 탐색하세요.

```bash
# 새 문서 추가 예시
docs/
├── javascript/
│   └── closure.md
├── react/
│   └── hooks.md
└── intro.md
```
