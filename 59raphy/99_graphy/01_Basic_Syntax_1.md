# Basic Syntax 1


## 표현식과 값
- 값은 가장 간단한 형태의 표현식
- 문장: 특정 동작을 지시하는, 실행 가능한 코드의 최소 단위
    - '값'이 남음


## 변수와 메모리
- 변수명은 숫자로 시작할 수 없다.
- 객체: *값, 타입, 주소 정보*를 묶은 개념
- 변수: 객체를 가리키는 이름


## Data Types
- 값의 종류와 그 값으로 할 수 있는 동작(연산)을 결정하는 속성
- 타입의 구성 요소: 값(피연산자)과 연산자
- 숫자는 더하기/ 문자열은 결합/ 숫자와 문자는 오류 발생

### 숫자형 데이터(Numeric Types)
1. 정수형(int)
2. 실수형(float)
    - 지수 표현법   ex) 1.23e9 = 1.23 * 10^9
    - 부동소수점 오차
        > from decimal import Decimal

        > Decimal('3.14')
3. 복소수(complex)

- 숫자형의 행동 : 산술 연산
- 연산자 우선순위
    1) 지수(**)
    2) 음수 부호
    3) 곱셈, 나눗셈, 몫 나눗셈, 나머지
    4) 덧셈, 뺄셈

- *불변 데이터*
- 진법 접두사
    - 2진수(0,1): 0b
    - 8진수(0~7): 0o
    - 16진수(0~9,a~f): 0x

### 시퀀스 타입(Sequence Types)
1. 순서, 나열 (정렬x)
2. 인덱싱  # my_sequence[0]
3. 슬라이싱  # my_sequence[start:stop:step]
    - [start:stop]
    - [:stop]
    - [start:]
    - [::step]
    - [::-1] : 역순 정렬
4. 길이  # len(my_sequence)
    - 다중 데이터
5. 반복

**문자열(str)**
- 여러 줄 문자열: """ 또는 '''
- 이스케이프 시퀀스 - 백슬래시(\)
    - \n : 줄바꿈
    - \t : 탭
    - \\ : 백슬래시
    - \' : 작은 따옴표
    - \" : 큰 따옴표
- 슬라이싱
    - my_str[::-1] : 역순 정렬
- *불변 데이터*
    - 재할당과 헷갈리지 않도록 주의 !
