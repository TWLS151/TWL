## 첫 풀이
```py
T = int(input())
for _ in range(1,T+1):
    result = []
    n = float(input())
    i = 1
    while n >= 0:
        if n == 0:
            print(*result)
            break
        if i == 13:
            print('overflow')
            break
        result.append(1)
        n -= 2**(-i)
        i += 1
        print(*result)
    else:
        result.append(0)
        n -= 2**(-i)
        print(*result)
        i += 1

print(*result)
```
### Issue-1
- issue
2진법을 어떻게 저장하는지 몰라서 list를 사용을 했다.
이후에 출력할 때 *를 사용해서 괄호를 풀면 되지 않을까라는 생각을 했다. 하지만 이 방식은 리스트의 item마다 공백이 추가된다.
- sol
result = ''라고 하자. 이후에 result += '0'이라고 하면 된다.
string 타입이어서 +라는 연산자를 사용하는게 어색해서 사용하지 못했다.

### Issue-2
- issue
첫 값 생각 못함.
처음부터 0.5이하인 값이 들어오면 '0'이 추가가 되어야 하지만 '1'이 추가되는 참사가 발생.
- sol

```py
T = int(input())
for _ in range(1,T+1):
    result = ''
    n = float(input())
    i = 1
    while n > 0:
        if i == 13:
            print('overflow')
            break
        if n >= 2**(-i):
            result += '1'
            n -= 2**(-i)
            i += 1
            print(*result)
        else:
            result += '0'
            n -= 2**(-i)
            i += 1
            print(*result)

print(*result)
```
